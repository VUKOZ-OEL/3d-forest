3D Forest 1.0
Development

There are several ways to add new functionality:

  1. The most easy way is to add new external plugin. External plugins are self contained. It is not necessary to touch any file outside of your plugin directory with exception of one cmake file.
    1. To start, one can copy and modify existing external plugin such as "src/plugins/classifyground" to new directory "src/plugins/myplugin".
    2. Add path to "src/plugins/CMakeLists.txt" to build the new plugin: 
              add_subdirectory(myplugin)
    3. External plugins will be visible from application main window.
  2. It is possible to add new command line tool to "src/apps/tools/mytool".
    1. Example command line tool "src/apps/tools/sandbox".
    2. Example command line tool with Qt "src/apps/tools/icon".
    3. Tools are not available from application main window.
  3. The third way is to add new internal plugin.
    1. To start, one can copy and modify existing internal plugin such as "src/plugins/projectfile" to new directory "src/plugins/myaction".
    2. Modify "src/gui/CMakeLists.txt" to build the new plugin into core: 
              file(GLOB_RECURSE SOURCES_MYACTION "../plugins/myaction/*.cpp")
        add_library
          ${SOURCES_MYACTION}
          ../plugins/myaction/resources/resources.qrc
        target_include_directories
          ../plugins/myaction
    3. Add field "guiPluginMyAction_" to "src/gui/GuiWindowMain.hpp". 
              class GuiPluginMyAction;
        GuiPluginMyAction *guiPluginMyAction_;
    4. Create instance in "src/gui/GuiWindowMain.cpp" constructor. 
              guiPluginMyAction_ = new GuiPluginMyAction(this);
    5. Internal plugins will be visible from application main window.
  4. The last and most difficult way is to modify core functionality in the following directories. This functionality is available in all plugins.
    • "src/core" build to "lib3dforestcore.dll"
    • "src/editor" build to "lib3dforesteditor.dll"
    • "src/gui" build to "lib3dforestgui.dll"

General:

  1. The main() 3D Forest application is in "src/apps/3dforest/3dforest.cpp".
  2. For development with Visual Studio Code, update ".vscode/settings.json" to contain new directories. For example, add "${workspaceRoot}/src/plugins/myaction" to "C_Cpp.default.includePath" include paths.
  3. Need some icon? Try to look into directory "3rdparty/icons".
